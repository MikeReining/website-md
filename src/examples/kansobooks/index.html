KansoBooks

KansoBooks is a live product launch website for accounting software, with trust-heavy copy, an interactive demo, waitlist CTA, pricing, comparison, and migration messaging.

Open live site Back to examples

Why It Matters

KansoBooks shows Website.md for a commercial product where wording, claims, proof, pricing, and user trust all matter. The agent needs more than page names. It needs operating rules.

Instruction Layers

`WEBSITE.md`

Routes, sections, CTA hierarchy, preview commands, validation checks, and publish approval rules.

`COPY.md`

Plainspoken financial-product voice, proof-first claims, and words to avoid.

`DEMO.md`

Interactive demo behavior, fake-data policy, walkthrough copy, and states that must remain safe.

`PRICING.md`

Pricing language, comparison rules, discount constraints, and claims requiring approval.

`MIGRATION.md`

QuickBooks migration positioning, source-of-truth rules, and accountant handoff language.

Agent Recipes

  • Update homepage positioning while preserving trust and compliance-sensitive claims.
  • Add a QuickBooks migration landing page.
  • Revise demo copy without implying real financial data is uploaded.
  • Check CTA links, metadata, mobile layout, and pricing language before deploy.
