Bitsboard

Bitsboard is a live education product website with app marketing, games, catalog, classes, accessibility, help, blog content, and App Store calls to action.

Open live site Back to examples

Why It Matters

Bitsboard is not a toy one-pager. It has a real product surface, public navigation, recurring content needs, SEO pressure, and accessibility promises. Website.md can act as the operating manual for agents changing that site.

Instruction Layers

`WEBSITE.md`

Routes, navigation, content locations, preview commands, validation, publishing rules, and agent recipes.

`BLOG.md`

Blog topics, post structure, educational tone, internal links, metadata, and publish workflow.

`CATALOG.md`

Rules for game pages, catalog pages, subjects, age ranges, and lesson metadata.

`APPSTORE.md`

Download CTA rules, app positioning, screenshot usage, and claims that must stay accurate.

`ACCESSIBILITY.md`

Accessibility commitments, inclusive language, alt text, and checks before publishing.

Agent Recipes

  • Publish a new educational blog post.
  • Add a new game or catalog landing page.
  • Update App Store messaging without changing unsupported claims.
  • Check important routes, metadata, sitemap, and mobile layout before deploy.
