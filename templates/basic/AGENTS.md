AGENTS.md tells agents how to work in your repo. WEBSITE.md tells agents how to operate your website.

Site instructions: see WEBSITE.md.

- `DESIGN.md`
- `COPY.md`
- `SEO.md`

Run checks before finishing:

```bash
npm run build
npm run check
```
