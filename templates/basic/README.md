# Basic Website.md Template

This is the intended shape of the first starter generated by:

```bash
npm create website-md@latest my-site
```

It should be a simple static-first website that an AI coding agent can modify quickly.

## Starter Contents

```txt
AGENTS.md
WEBSITE.md
DESIGN.md
COPY.md
SEO.md
package.json
content/pages/home.md
src/
public/
```

## Required Commands

```bash
npm run dev
npm run build
npm run check
```

## Agent Prompt

```txt
Use WEBSITE.md, DESIGN.md, COPY.md, and SEO.md to customize this site for my business. Preview locally and run checks before you finish.
```
