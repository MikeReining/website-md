# DESIGN.md

## Visual Direction

Clean, direct, and trustworthy. The site should feel useful within the first viewport.

## Colors

- Background: white or near-white.
- Text: dark neutral.
- Accent: one restrained action color.
- Avoid decorative gradients.

## Typography

Use system fonts. Keep headings clear and body text readable.

## Layout

- Put the primary offer in the first viewport.
- Keep sections scannable.
- Use cards only for repeated items.
- Check mobile before finishing.

## Components

- Buttons for primary actions.
- Code blocks for commands.
- Simple navigation.
