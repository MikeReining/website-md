# SEO.md

## Purpose

Help AI agents keep the site discoverable without creating spam.

## Rules

- Preserve public URLs.
- Give every page a title and meta description.
- Use canonical URLs.
- Link new pages from at least one existing page.
- Keep claims accurate.
- Avoid keyword stuffing.

## Required Files

- `src/robots.txt`
- `src/sitemap.xml`
- `src/llms.txt`
