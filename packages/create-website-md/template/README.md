# Website.md Starter

This is a static-first website operated by AI using Markdown instruction files.

## Start

```bash
npm run dev
```

Open:

```txt
http://localhost:4177
```

Keep the dev-server terminal open while previewing.

## Agent-First Setup

Paste this into an AI coding agent:

```bash
npm create website-md@latest my-site
```

The agent should create the folder, enter it, read the instruction files, run the dev server, and give you the localhost URL.

## Ask Your AI Agent

```txt
Use WEBSITE.md, DESIGN.md, COPY.md, and SEO.md to customize this site for my business. Preview locally and run checks before you finish.
```
